## Projeto

Para rodar o projeto localmente é necessário ter o node instalado na sua máquina e após isso executar os seguintes comandos após a clonagem do projeto:

```javascript
npm install
npm run dev
```

## Tecnologias utilizadas:

- Vite
- React
- SCSS
